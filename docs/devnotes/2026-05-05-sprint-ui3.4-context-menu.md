## Context & Goal

Add right-click context menu to media thumbnails with actions: Copy, Delete, Size (Big/Medium/Small), Hide/Show, Show/Hide Label, Show/Hide Date, Show/Hide Time, and Rename.

## Decisions & Alternatives

- **Shared SetupContainerInteractions method:** Extracted hover effects + drag handlers + context menu into one method called by both CreateImageContainer and CreatePlaceholderContainer. Eliminates duplicate code.
- **MaterialDesignThemes.Wpf.PackIcon for menu icons:** Uses full namespace qualification to avoid adding a using statement (minimal diff).
- **EditLabel placeholder:** Shows DialogHelper message — full inline editing deferred to UI-3.5.
- **MediaReferences getter fix:** Changed from creating new MediaReference instances to reading existing ones from container tags. This preserves v3 metadata (Label, ThumbnailSize, IsHidden, etc.) during save.
- **MediaReferences setter fix:** Now passes a Dictionary<string, MediaReference> to LoadImagesFromFiles so metadata is preserved when loading saved notes.

## Implementation Notes

### New Methods (all in MediaSection.xaml.cs)

| Method | Purpose |
|--------|---------|
| `CreateIcon(PackIconKind)` | Creates 16x16 MaterialDesign icon for menu items |
| `SetupContainerInteractions(Grid, string, MediaReference)` | Hover + context menu + drag handlers |
| `CopyImageToClipboard(string)` | Copies image to system clipboard via cache or fresh load |
| `DeleteImageFromMenu(Grid, MediaReference)` | Deletes file from disk + removes container |
| `ChangeThumbnailSize(Grid, MediaReference, int)` | Updates size + rebuilds single container |
| `ToggleHidden(Grid, MediaReference)` | Swaps image for placeholder dots or rebuilds |
| `ToggleLabelVisibility(Grid, MediaReference)` | Toggles ShowLabel + updates TextBlock |
| `ToggleDateVisibility(Grid, MediaReference)` | Toggles ShowDate + updates timestamp |
| `ToggleTimeVisibility(Grid, MediaReference)` | Toggles ShowTime + updates timestamp |
| `UpdateTimestampText(Grid, MediaReference)` | Replaces row-2 TextBlock with reformatted timestamp |
| `EditLabel(Grid, MediaReference)` | Placeholder — shows "not yet available" dialog |
| `RebuildSingleContainer(Grid, MediaReference)` | Replaces one container in-place at same panel index |

### Refactored Methods

- `CreateImageContainer` — replaced inline hover/drag with `SetupContainerInteractions` call
- `CreatePlaceholderContainer` — replaced inline hover/drag with `SetupContainerInteractions` call
- `LoadImagesFromFiles` — added optional `Dictionary<string, MediaReference>?` parameter
- `MediaReferences` getter — reads from container tags instead of creating fresh instances
- `MediaReferences` setter — passes metadata dict to `LoadImagesFromFiles`

### Bug Fix During Implementation

`ItemsControl.ItemsControlFromItemContainer(container)` returns `ItemsControl`, not `WrapPanel`. Fixed by using `container.Parent as WrapPanel` instead.

## Testing & Acceptance

- `dotnet build` — 0 errors, 3 pre-existing warnings
- Right-click any thumbnail → context menu appears with all items
- Each menu action updates the MediaReference on the container tag
- `OnMediaChanged` fires after each action (triggers auto-save)

## Performance & Limits

- Context menu is created per-container (not shared). Acceptable for typical thumbnail counts.
- `RebuildSingleContainer` creates a new placeholder + triggers async load. Slightly heavier than a direct resize, but simpler and correct.
- `CopyImageToClipboard` checks cache first to avoid redundant file I/O.
